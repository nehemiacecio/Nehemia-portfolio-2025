# Theme Color Palettes

## 1) Light Mode Color Palette
Use these colors as the primary theme for Light Mode:

- **Primary (Teal):** `#3F9AAE`
- **Secondary (Mint):** `#79C9C5`
- **Base / Background (Cream):** `#FFE2AF`
- **Accent / CTA (Coral):** `#F96E5B`

**Suggested Usage:**
- Page background: `#FFE2AF`
- Main headers / navigation / key UI: `#3F9AAE`
- Cards / panels / soft sections: `#79C9C5`
- Primary buttons / highlights / alerts: `#F96E5B`

---

## 2) Dark Mode Color Palette
Use these colors as the primary theme for Dark Mode:

- **Base / Background (Dark):** `#2C272E`
- **Primary (Purple):** `#753188`
- **Secondary (Orange):** `#E59934`
- **Accent (Green):** `#9AE66E`

**Suggested Usage:**
- Page background: `#2C272E`
- Main headers / navigation / key UI: `#753188`
- Cards / badges / warm highlights: `#E59934`
- Primary buttons / success / active states: `#9AE66E`
